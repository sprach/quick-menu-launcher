# QikMenu (quick-menu-launcher)
[English](README.md) | **한국어** | [日本語](README_JA.md)

Rust로 작성된 가벼운 윈도우 시스템 트레이 애플리케이션입니다. INI 파일에 정의된 메뉴를 통해 자주 사용하는 프로그램이나 웹사이트를 빠르게 실행할 수 있습니다.

## 개발 배경
이 앱은 다음과 같은 필요에 의해 제작되었습니다.
- **옵시디안(Obsidian) Vault 관리**: 용도에 따라 나뉘어진 여러 옵시디안 저장소(Vault)를 필요할 때마다 간편하게 선택해서 열고 싶었습니다.
- **휴대 장비의 한계 극복**: 평소 Stream Dock(Stream Deck)에 단축키를 등록해 사용했지만, 장비를 휴대할 수 없는 환경에서도 시스템 트레이를 통해 동일한 기능을 사용하고 싶었습니다.
- **작업 표시줄 그룹화**: 기존 윈도우 작업 표시줄을 유지하면서도, 특정 앱들을 그룹으로 묶어 깔끔하게 관리하고 선택 실행하고 싶었습니다.
- **범용성 확장**: 옵시디안뿐만 아니라 자주 사용하는 다른 프로그램들도 쉽게 등록하여 호출할 수 있도록 기능을 확장했습니다.

## 설치 및 실행 방법
1. 다운로드 받은 `7z` 압축 파일을 원하는 폴더에 풉니다.
2. `QikMenu.exe` 파일이 있는지 확인합니다.
3. 같은 폴더에 `QikMenu.ini` 설정 파일이 있어야 합니다.
4. `QikMenu.exe`를 더블 클릭하여 실행합니다.

## 사용법
프로그램이 실행되면 윈도우 작업 표시줄의 시스템 트레이 영역(오른쪽 하단)에 아이콘이 나타납니다.

![트레이 메뉴 스크린샷](refs/cap-launchTrayMenu.jpg)

1.  **메뉴 열기**: 트레이 아이콘을 마우스로 클릭(좌/우 클릭 모두 가능)하면 위와 같은 메뉴가 나타납니다.
2.  **프로그램 실행**: 메뉴 목록에서 원하는 항목을 클릭하면 해당 프로그램이나 웹사이트가 실행됩니다.
3.  **환경 설정 편집**:
    - 메뉴 맨 아래쪽의 **"환경 편집"**을 클릭합니다.
    - `QikMenu.ini` 파일이 메모장(또는 기본 편집기)으로 열립니다.
    - 원하는 항목을 추가하거나 수정합니다 (아래 설정 섹션 참고).
    - 파일을 저장하고 닫습니다.
4.  **설정 적용 (새로 고침)**:
    - 설정을 수정한 후, 메뉴의 **"환경 다시 읽기"**를 클릭합니다.
    - 프로그램을 재시작하지 않아도 변경된 내용이 즉시 메뉴에 반영됩니다.
5.  **종료**: 메뉴의 **"종료"**를 클릭하면 프로그램이 완전히 종료됩니다.

## 메뉴 호출 방법
메뉴를 여는 방법은 두 가지가 있습니다:

1.  **트레이 아이콘 클릭**:
    -   윈도우 작업 표시줄의 시스템 트레이에 있는 **QikMenu 아이콘**을 좌클릭 또는 우클릭하면 메뉴가 열리거나 닫힙니다.

2.  **전역 단축키 (Global Hotkey)**:
    -   윈도우 어디서든 미리 정의된 단축키(예: `[Alt]+/`)를 누르면 메뉴가 팝업됩니다.
    -   이 단축키는 `QikMenu.ini` 파일의 `[env]` 섹션에서 설정할 수 있습니다 (아래 설정 참고).

## 윈도우 시작 시 자동 실행
윈도우가 시작될 때마다 이 프로그램이 자동으로 실행되게 하려면:

1. 키보드의 `Win` 키 + `R` 키를 눌러 **실행** 창을 엽니다.
2. `shell:startup` 이라고 입력하고 확인을 누릅니다. **시작프로그램** 폴더가 열립니다.
3. `QikMenu.exe` 파일에 마우스 오른쪽 클릭을 하고 **"바탕 화면에 바로 가기 만들기"** 등을 이용해 바로 가기 아이콘을 만듭니다.
4. 만들어진 **"바로 가기"** 파일을 아까 열린 **시작프로그램** 폴더로 복사하거나 이동합니다.
5. 이제 윈도우 부팅 시 프로그램이 자동으로 실행됩니다.

## 설정 파일 (`QikMenu.ini`)
```ini
[global]
# 사용 가능한 언어: ko (한국어), en (영어), ja (일본어)
locale=ko

[env]
# 'hotkey'는 QikMenu가 실행된 상태에서 팝업 메뉴를 띄우는 단축키
# 특수키: [Alt], [Shift], [Ctrl], [Win], [F1]~[F12] 등
# 예: [Alt]+/ (Alt 키와 / 키를 동시에 누름)
hotkey=[Alt]+/

[apps]
# 형식: 표시이름=명령어
# 참고: 경로에 공백이 포함된 경우 따옴표로 묶어주세요:
# 예: 내 프로그램="C:\Program Files\MyApp\app.exe"
# 옵시디안
Obsidian MyVault1=obsidian://open/?vault=MyWorks1
Obsidian MyVault2=obsidian://open/?vault=MyWorks2
Obsidian MyVault3=obsidian://open/?vault=MyWorks3
# 기타
명령 프롬프트=cmd
네이버=https://naver.com
메모장=notepad.exe
```
